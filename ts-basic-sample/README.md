# Typescript Basic Sample

## How to use

- Prepare

  ```bash
  git clone https://github.com/SPONGE-JL/CodeLab-Typescript.git
  cd ts-basic-sample && npm install
  ```

- Running

  ```bash
  npm run study {chNum}
  ```

- Chapter number list

  | chNum | Keyword                  |
  |:-----:|:-------------------------|
  |  `03` | Object, Interface, Class |
